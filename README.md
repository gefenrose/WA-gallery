# יוצר אלבומי WhatsApp

אפליקציית Vite, React ו-TypeScript מקומית להפיכת קובץ ZIP שיוצא מקבוצת WhatsApp לאלבום משפחתי אישי בקובץ HTML.

האפליקציה רצה כולה בדפדפן. היא לא מתחברת ל-WhatsApp, לא משתמשת ב-API של WhatsApp, לא מעלה קבצים ולא דורשת שרת.

## התקנה והרצה

```bash
npm install
npm run dev
npm run build
```

## ייצוא צ'אט מ-WhatsApp

1. פתחו את קבוצת ה-WhatsApp.
2. לפני הייצוא, עברו בצ'אט הקבוצה וודאו שהתמונות והסרטונים שאתם רוצים באלבום מופיעים בצ'אט עצמו וירדו למכשיר. אם רואים סמל הורדה על תמונה או סרטון, פתחו או הורידו אותם קודם.
3. פתחו את פרטי הקבוצה או התפריט.
4. בחרו ייצוא צ'אט.
5. בחרו לכלול מדיה.
6. שמרו או שתפו את קובץ ה-ZIP שיוצא.
7. ייבאו את קובץ ה-ZIP לאפליקציה.

אם המדיה לא הורדה או לא מוצגת בצ'אט לפני הייצוא, WhatsApp עלולה לא לכלול אותה בקובץ ה-ZIP, והאפליקציה לא תוכל להציג אותה באלבום.

## יצירת אלבום אישי

1. ייבאו את קובץ ה-ZIP לאפליקציה.
2. בדקו את התמונות והסרטונים, תקנו כיתובים אם צריך, וסננו לפי שולח/ת או כיתובים חסרים.
3. לחצו על **ייצוא אלבום אישי**.
4. הדפדפן יוריד קובץ ZIP חדש שנקרא לפי שם קבוצת הצ'אט, ככל שניתן לזהות אותו משם הקובץ המקורי.
5. פתחו את ה-ZIP החדש וחלצו אותו לתיקייה במחשב או בטלפון.
6. פתחו את הקובץ `index.html` מתוך התיקייה שחולצה. חשוב להשאיר את תיקיית `media` לידו, כי שם נמצאות התמונות והסרטונים.
7. בתוך האלבום, לחצו על תמונה או סרטון כדי לפתוח תצוגה גדולה. אפשר לעבור קדימה ואחורה בין כל פריטי המדיה, והכיתוב מופיע מתחת למדיה.

## הערות

- האפליקציה מחפשת את `_chat.txt` או קובץ צ'אט אחר מסוג `.txt` בתוך ה-ZIP.
- תמונות וסרטונים מוצגים באמצעות קישורי `URL.createObjectURL` מקומיים, והקישורים הקודמים מבוטלים כשנטען ZIP חדש.
