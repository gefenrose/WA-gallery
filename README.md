# יוצר אלבומי WhatsApp

אפליקציית Vite, React ו-TypeScript מקומית להפיכת קובץ ZIP שיוצא מקבוצת WhatsApp לאלבום משפחתי להדפסה.

האפליקציה רצה כולה בדפדפן. היא לא מתחברת ל-WhatsApp, לא משתמשת ב-API של WhatsApp, לא מעלה קבצים ולא דורשת שרת.

## התקנה והרצה

```bash
npm install
npm run dev
npm run build
```

## ייצוא צ'אט מ-WhatsApp

1. פתחו את קבוצת ה-WhatsApp.
2. פתחו את פרטי הקבוצה או התפריט.
3. בחרו ייצוא צ'אט.
4. בחרו לכלול מדיה.
5. שמרו או שתפו את קובץ ה-ZIP שיוצא.
6. ייבאו את קובץ ה-ZIP לאפליקציה.

## הערות

- האפליקציה מחפשת את `_chat.txt` או קובץ צ'אט אחר מסוג `.txt` בתוך ה-ZIP.
- תמונות וסרטונים מוצגים באמצעות קישורי `URL.createObjectURL` מקומיים, והקישורים הקודמים מבוטלים כשנטען ZIP חדש.
- ההדפסה משתמשת בדיאלוג ההדפסה של הדפדפן, כך שאפשר לבחור שמירה כ-PDF.
